# 🍽️ CampusBite

## 📖 Project Description
CampusBite is a UI/UX prototype of a campus food ordering application developed as part of the CodeAlpha UI/UX Design Internship. This project demonstrates the user interface and user journey for ordering food inside a college campus. It is a demo project created for learning and design purposes and is not connected to a real ordering system.

---

## ✨ Features

- User Login
- User Signup
- Home Screen
- Restaurant Browsing
- Menu Exploration
- Add to Cart
- Cart Management
- Order Summary
- Order Tracking
- Responsive UI Design

---

## 🛠️ Technologies Used

- React.js
- JavaScript (JSX)
- HTML
- CSS
- Lucide React Icons
- StackBlitz
- VS Code
